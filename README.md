#UML to C++ Generator
Overview:
This project converts XML class diagrams into C++ header files.
It parses class information such as:
_Class name
_Attributes
_Methods
_Inheritance
Then it generates corresponding C++ class code automatically.
##Current Progress:
Created the 'ClassModel' .

##Planned Features
_XmL parsing
_Code generation
_Support for inheritance
_Error handling
